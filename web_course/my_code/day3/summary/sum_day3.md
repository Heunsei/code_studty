# BOOTSTRAP
- CSS 프론트엔드 프레임워크
- 미리 만들어진 디자인 요소들을 제공하여 웹 사이트를 빠르고 쉽게 개발

## 기본 사용법
- 모든 스타일링은 스타일로 사용
- 브라우저의 기본 루트 픽셀은 16px
```
<p class="mt-5">Hello, world!</p>
{margin}{top} - {size}
```
